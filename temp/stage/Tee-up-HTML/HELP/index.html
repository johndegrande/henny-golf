Set Up

Welcome, this is a

HTML template (not Wordpress)

so all the files are fully operational to be used now. Open the folder with a text editor and replace our content with your content, you'll find out here all the things that you need to know to build your perfect website.

Watch out to the PHP

Some features needs PHP, if you want to use the contact form you need PHP (see #ContactPage section).

Boxed Layout

You can enable the boxed layout on any page by adding the class Boxed to the DIV #page:

<div id="page"> -> <div id="page" class="boxed">

Animations

There is 5 animations (classes) for the content in this template

  • animate_top_bottom: Animation from top to bottom.
  • animate_bottom_top: Animation from bottom to top.
  • animate_left_right: Animation from left to right.
  • animate_right_left: Animation from top to bottom.
  • animate_fade: Animation fade In.

You can add one of those classes to any HTML element, it'll be animated by CSS3. You can customize the animations into style.css at the end there is a section for the animations. If you want to remove the animations, just delete the ANIMATIONS section in CUSTOM.JS.Then, note than under 992px of screen width, the animations are disabled

Some CSS Class Really Useful

  • highlight: highlight a text with a yellow background (I advice you to use it with a span tag).
  • dropcap: Just a dropcap letter at the beginning of a paragraph.
  • center: center the text inside a container (like a p tag)
You can also use all the Bootstrap's class

Pattern for the background of some sections

Same as for the footer, see Footer section > Background

Fonts

This template use "PlayFair Display" and "Open Sans". But if you want to change the fonts:

  • Go to google fonts website, then choose your font, go to “quick use” and copy this line (ctrl + c),
  • past this line in the header of the template (delete the previous).
  • Go to style.css (line 65+). All the elements with font-family are combined at the beginning of the file, just replace the previous font with the new one.

Icons & Bootstrap

Icons

About the icons we're using an icon font, You can see the icons library there -> ASSETS / ICONS / DEMO.HTML This file is really important, you should keep it open in an other tab of your browser during the development. In deed, you'll find out there a css class for each icon. Then when you found that class, just create a html tage like this -> <i class="icon-yourIconName" > </i>. An icon gonna appear ! Note that you can use it anywhere in your html code, also and especially in the buttons.

Bootstrap

Bootstrap is included with this template. That mean that you can use any functionality, components from Bootstrap(for a size optimization, all the js plugins from bootstrap aren't included with this template; only alert.js and dropdown.js). Please note that some of them are designed, changed for the design of this template. All changes are in style.css, the Bootstrap file is untouched

Header (logo, navigation)

The header of this template is the part upper with the logo, menu and background image

Logo

Your logo should be a PNG image file. Just call it "logo" and copy it in the image folder of the template. Moreover, you can change the logo size which is by default 300 px in style.css (header part -> #logo)

Menu

The menus are managed with the HTML. So if you want to edit your menu, please do the change in the other files too. The menu is a big list in the top of the file (id="navigation"). Watch out ! In the Member's Area files, which are in an external folder, you need to change the path of the link, just add "../" before a new page created in the main template.

Add item to your menu

Just copy the following line (set the link and the name of the page):

<li "><a href="your link">Page Name</a>

Dropdown Menu

<li class="with-submenu" >
    <a href="link.html">Name</a>
    <ul class="submenu">
      <li><a href="link">Sub item 1</a></li>
      <li><a href="link">Sub item 2</a></li>
    </ul>
</li>

Pages background (Header)

You may notice that there is a independent header background for each page. For the home page it's a slider (see "Home" section below). About the others, you can change it directly in the HTML part of each page file. The html tag is below the menu, it looks like:

<!-- BACKGROUND HEADER -->
<div id="background" class="small" style="background-image: url('images/slider/1.jpg');"></div>

So you just have to add the path to your background image (by default it use the images used for the slider), you can also replace the background-image property by the background-color property.

Home

The home page is the most important page of the template. So we created it in the way to display the most elements about your Golf in an elegant way. Therefore, it's devided in differents sections and with a slider to display some awesome pictures .

Slider

There is a folder for the images caller "slider" into the "images" folder. You can add new slider within the div#background. the structure should like:

<li data-transition="slideleft" data-slotamount="1">
	<img src="images/slider/2.jpg" alt="Image Slide">
	<!-- TEXT CAPTION (CAN BE DELETED) -->
	<div class="caption lfr"  data-x="0" data-speed="300" data-start="1000">A caption to describe your photo.</div>
</li>

Single Event

You have to know 2 things about the single event page which are really important: the timer and the map:

The Timer

To configure your timer you just need a date, go to the end of the event page file, all is explained in the comments.

Google Map

The map is initialized in single-event.html(php). For the map we need the position of your business. To find it go to: Itouchmap. A simple click from the site map on the location of your business will give you the longitude and latitude (for example: 38.941182, -46.625977), copy the result in the bottom of the page "var myLatlng = new google.maps.LatLng(48.858000, 2.294600);" . You're free to edit the settings of the map with the Google API. The javascript code which enable the map and where you have to set your location is into the footer where all the js files are included.

Gallery Page

You can see the gallery in the home page and in the gallery page. The structure is the same. You can add there any photos, you should put them into the "gallery" folder which is within the "images" folder.

Add new item

<!-- ITEM GALLERY -->
<div class="gallery-item item">
	<a href="images/gallery/6.jpg" data-fancybox-group="group1" class="fancybox">
		<span> Image Title </span>
		<img src="images/gallery/6.jpg" width="300" height="200" alt="Image gallery">
	</a>
</div>
<!-- END ITEM -->

Filtering

The filter buttons works with classes to the HTM gallery item DIV. Each button call a class, so you have to add this called class to all the item gallery which are inside it. This is pretty hard to understand, so I recommand you to take a look to the plugin documentation about that -> Isotope Documentation

Contact Page

Maybe you've seen, there is 2 contact files: contact.HTML is just for the HTML template the form isn't working at all whereas index.PHP contained the sending code of the email. If you'll delete the form don't use it.

Form

To use the form please, delete contact.html and keep contact.php. So go to the top of the file you'll see a variable called "$yourEmail", fill it. It'll works now.

Issues:

You do not receive alerts or error sends after sending:

  • Check if you're on contact.php and not contact.html, then delete contact.html from your server

You see a warning of success but you receive nothing:

  • Check if your provider accepted the sending of emails
  • Check if your email is correctly set in contact.php line 3
  • Check your mailbox again...

New Page

Create a New Page

Please, follows these steps

  1. Open a new ".html" file in your editor
  2. Copy the containing of new-page.html
  3. Replace all the "Page name" or "title here" with your page name
  4. Write your content and don't forget to try to add elements such as: buttons, columns, h3....
  5. Open each pages of your website
  6. Update your menu with this new page (see: "header" section "menu")

Footer

Background

The background of the footer is a pattern called "footer_bg" in the "images" folder (footer_bg@2X is the one used for the high resolution screen). If you want to change it:

  1. Go to Subtle Patterns
  2. Choose and download your favorite
  3. Copy/paste both images files (normal and @2x) and rename them into "footer_bg"
  4. Sometimes you have also to change the background size property in style.css
